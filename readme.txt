In the store for the calendar to display the holiday shop in the widget.

== Description ==

店舗向けのカレンダーで、店休日をウェジットに表示するプラグインです。

=== Features of the this version ===
* 曜日を指定して、店休日に設定できます
* 第何曜日を指定して、店休日に設定できます。
* 長期店休日を指定して、店休日に設定できます。
* ショートコードも対応しています。

== Installation ==

1. Upload `shop-calendar.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Appear under Settings menu.
4. Select the days of the week and date of the shop holiday.
5. Click save.
